# betaStore

Steps to run:

- git clone this repo

- Make sure the rails version you have installed is `5.2.0`

- run `bundle install`

- run `rails db:create`

- run `rails g spree:install --user_class=Spree::User`

- run `rails g spree:auth:install`

- run `rails g spree:auth:install`

- run `rails server`

Should work
